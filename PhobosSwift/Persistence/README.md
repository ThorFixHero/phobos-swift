PhobosSwiftPersistence
================
# Features
- Realm: Realm封装


# Requirements
- iOS 10.0 or later
- Xcode 11.0 or later
- cocoapods 1.10.0 or later


# How To Use
- swift

```
import PhobosSwiftPersistence

let realm = Realm.pbs.makeRealm()
```

# Installation
## Podfile

```
platform :ios, '10.0'
use_frameworks!
pod 'PhobosSwiftPersistence', '~> 0.1.0'
```
