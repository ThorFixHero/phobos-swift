PhobosSwiftSplash
================
# Features
- APP进入后台时默认展示页


# Requirements
- iOS 10.0 or later
- Xcode 11.0 or later
- cocoapods 1.10.0 or later


# How To Use
- swift

```

import PhobosSwiftSplash

PBSSplash.shared.configuration
```

# Installation
## Podfile

```
platform :ios, '10.0'
use_frameworks!
pod 'PhobosSwiftSplash', '~> 0.1.0'
```
